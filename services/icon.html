Description

An icon/token commission includes a set of portraits of one or multiple characters. For my fellow TTRPG players, they may come with a special, fancy token frame for use in virtual tabletops such as Roll20 or Foundry!

Such a commission may include up to ten portraits of either the same or multiple characters, and up to three different frames per character.
You can see on the provided examples that I like to play a lot on symbolics, so don't hesitate to request the wildest things for token frames!

Pricing

All prices are given in USD.

Base price per portrait: $20 (simple/neutral expressions), $30 (complex expressions, or extra stylization)
Only solid colors allowed as backgrounds (free)

Base price per token frame: $5 (simple designs), $10 (intricate designs)
Up to three frames allowed for each individual character, even if commissioning multiple portraits of the same character.
Alternatively, if you already have an artwork of your character, I can directly tokenize it, applying the individual rates for token frames.
